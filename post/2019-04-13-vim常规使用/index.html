基础操作

  • i、a - 光标处、后插入
  • o、O - 光标行后、前一行插入
  • x - 删除光标后一个字符
  • r、R - 替换一个、多个字符
  • dd - 删除当前行
  • p - 粘贴
  • yy - 拷贝当前行
  • u - undo
  • ctrl + r - redo
  • ctrl + b、ctrl + f - 上一页、下一页
  • ctrl + u、ctrl + d - 上半页、下半页
  • ctrl + y、ctrl + e - 向上、下滚动一行
  • :s /old/new 、:s /old/new/g- 当前行第一个、所有old字符换为new字符
  • :% s/old/new/g - 整个文件替换字符old为new
  • esc - 退出编辑模式，回到normal模式

光标移动

  • hjkl - 分别对应←↓↑→
  • w、W - 到下一个单词开头，W到下一个单词（遇到blank后）开头
  • e、E - 到单词结尾，E到单词（直到遇到blank）结尾
  • b、B - 到单词开头，B到单词（直到遇到blank）开头
  • 0 - 回行头
  • ^|_ - 到非blank字符行头
  • +和- - 移动到下一行开头、上一行开头
  • $ - 到行尾
  • g_ - 到非blank字符行尾
  • gg|1G|:1 - 回到文件第一行
  • G - 到文件最尾一行
  • NG|:N|Ngg - 到第N行
  • *、# - 匹配光标当前单词上一个(#)和下一个(*)
  • % - 括号间移动，匹配(){}[]，需把光标放到括号上
  • fh - 移动光标到下一个字符为h的地方，Nfh为下N个
  • th - 移动光标到下一个字符为h的前一个字符，Nth为下N个
  • z - 将光标行移动到屏幕顶端
  • z. - 将光标行移动到屏幕中间
  • H、M、L - 将光标移动到屏幕顶端、中间、底部
  • N| - 移动到当前行的第N列
  • (、) - 移动到句子开头、下一句子开头
  • {、} - 移动到段落开头、下一段落开头

执行多次

N 执行N此command

  • . - 执行上次的命令
  • 2. - 执行2此上次命令
  • 2dd - 删除2行
  • 5p - 粘贴5行
  • 10yy - 复制10行
  • 7i - 写入字符pdc，则esc退出后会重复写入7个pdc
  • 9o | 9O - 写入字符enter，则esc退出后会重复写入9行enter字符

中级操作

  • yg_ - 拷贝行开始到结尾，不带blank字符，也有类似的yG
  • ye - 拷贝当前位置到单词结尾，还有类似的yw，yE，yW
  • y2/hugo - 拷贝两个字符hugo之间的字符
  • de - 剪切当前位置到单词结尾，类似的dE
  • dw - 剪切当前位置到下一个单词开头，类似的dW
  • cw - 删除从光标处到下一个单词结尾的字符，并进入insert
  • D - 剪切光标处到行结尾
  • v - 可视化选择，d删除，y拷贝，gU变为大写，gu变为小写

其他

  • /pattern、?pattern - 搜索pattern字符串，从上而下、下而上查找。n可查找下一个，N为上一个
  • :set ignorecase|:set ic、:set noignorecase|:set noic - 设置忽略、不忽略大小写，搜索时是大小写敏感的
  • ctrl + g - 显示当前行信息
  • :set nu|:set number、:set nu!|set number!|:set nonu|:set nonumber - 显示、取消行号
  • :syntax enable|:syntax on、:syntax clear|:set syntax=off|:syntax off - 打开、关闭高亮功能(彩色语法)
  • :set hlsearch、:set nohlsearch - 设置高亮、取消高亮显示搜索结果
  • :! command - 运行外部命令

文件操作

  • :q - 退出
  • :q! - 强制退出，不保存修改
  • :qa!- 强制退出正在编辑的所有文件，不保存修改
  • :w、:w other_file - 保存、另存为other_file
  • :wq | :x | ZZ - 保存并退出，ZZ不需要按回车
  • :e filepath - 编辑文件
  • :bn、:bp - 同时打开很多文件时，切换下一个(bn)跟上一个文件(bp)

高级操作

文件加密解密

  • :set key=password - 加密文件，password为密码
  • :set key= - 不加密文件
