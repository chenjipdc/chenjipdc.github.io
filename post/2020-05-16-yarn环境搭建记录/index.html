首先应该先起hdfs

单点yarn

首先进入目录：/opt/bigdata/apache-hadoop-2.6.5-local/etc/hadoop/

修改配置文件mapred-site.xml

<property>
    <name>mapreduce.framework.name</name>
    <value>yarn</value>
</property>

修改配置文件yarn-site.xml

<!-- shuffle 洗牌 -->
<property>
    <name>yarn.nodemanager.aux-services</name>
    <value>mapreduce_shuffle</value>
</property>

启动

start-yarn.sh

高可用集群HA

节点规划

  • node00 resource manager
  • node01 node manager
  • node02 node manager + resource manager
  • node03 node manager

修改配置文件mapred-site.xml

<property>
    <name>mapreduce.framework.name</name>
    <value>yarn</value>
</property>

这里yarn也可以改为local，本地debug的时候使用

修改配置文件yarn-site.xml

<!-- shuffle 洗牌 -->
<property>
    <name>yarn.nodemanager.aux-services</name>
    <value>mapreduce_shuffle</value>
</property>

<!-- 开启rm高可用 -->
 <property>
   	<name>yarn.resourcemanager.ha.enabled</name>
   	<value>true</value>
 </property>
 <!-- 设置rm集群id，区别其他集群 -->
 <property>
   	<name>yarn.resourcemanager.cluster-id</name>
   	<value>chenjipdc</value>
 </property>
 <!-- rm关系映射 -->
 <property>
   	<name>yarn.resourcemanager.ha.rm-ids</name>
   	<value>rm1,rm2</value>
 </property>
 <property>
   	<name>yarn.resourcemanager.hostname.rm1</name>
   	<value>node00</value>
 </property>
 <property>
   	<name>yarn.resourcemanager.hostname.rm2</name>
   	<value>node02</value>
 </property>
 <property>
   	<name>yarn.resourcemanager.zk-address</name>
   	<value>node01:2181,node02:2181,node03:2181</value>
 </property>

分发配置文件到其他节点

scp mapred-site.xml yarn-site.xml hadoop@node01:`pwd`
scp mapred-site.xml yarn-site.xml hadoop@node02:`pwd`
scp mapred-site.xml yarn-site.xml hadoop@node03:`pwd`

启动/关闭yarn

# node00启动rm跟其他nm
start-yarn.sh
# node02 启动rm
yarn-daemon.sh start resourcemanager

# node00关闭yarn
stop-yarn.sh
yarn-daemon.sh stop resourcemanager

其他

获取rm状态

yarn rmadmin -getServiceState rm1
yarn rmadmin -getServiceState rm2

手动切换为standby，前提未设置自动failover

yarn rmadmin -transitionToStandby rm1

yarn访问地址：

  • http://node00:8088/cluster/cluster
  • http://node02:8088/cluster/cluster
  • yarn
